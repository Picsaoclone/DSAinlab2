# Stack and Queue Implementations in C++

This repository contains implementations of stack and queue data structures using singly linked lists in C++. The exercises demonstrate fundamental operations and provide a menu-driven interface for user interaction.

## Features

- Stack:
  - Push an element onto the stack.
  - Pop an element from the stack.
  - Peek at the top element of the stack.
  - Check if the stack is empty or full.
  - Display all elements in the stack.

- Queue:
  - Enqueue an element into the queue.
  - Dequeue an element from the queue.
  - Peek at the front element of the queue.
  - Check if the queue is empty or full.
  - Display all elements in the queue.
  - Retrieve the rear element of the queue.

## Stack Implementation

The stack is implemented using a singly linked list. The following operations are supported:

1. **Push**: Add an element to the top of the stack.
2. **Pop**: Remove the top element from the stack.
3. **Peek**: View the top element without removing it.
4. **isEmpty**: Check if the stack is empty.
5. **isFull**: Check if the stack is full (always returns false since linked lists can grow dynamically).
6. **Display**: Print all elements in the stack.
